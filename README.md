# xlinkote

## 简介

为了整理错乱的考试知识点而随手搞得一个简单的笔记系统

不纯粹，我希望尽可能把好的笔记功能都加入

无需注册，不存在官方云存储，只使用 WebDAV 代替

[在线使用](https://xlinkote.vercel.app/)

## 功能

-   [ ] 画布
    -   [x] markdown 编辑
    -   [x] 无限画布
-   [ ] 保存
    -   [x] 本地保存
    -   [x] WebDAV
-   [ ] 元素(组件)
    -   [x] markdown 元素(支持换行渲染)
    -   [x] 数学公式($\LaTeX$数学公式,图形化输入)
    -   [x] 画板
        -   [ ] 粗细
        -   [ ] 颜色
        -   [ ] 图层
    -   [ ] 外部导入
        -   [x] 拖拽
        -   [ ] 粘贴
        -   [x] HTML 转 markdown
    -   [ ] 录音
    -   [ ] 思维导图
